## alphavantage_TOOL_LIST

### What this tool is for
The available Alpha Vantage API tools along with their names and descriptions. It is useful as the discovery entry point for this server before selecting a specific tool to inspect or call.

---

### Used parameters

This tool takes no parameters.

---

### JSON input alternatives

```json
{
  "intent": "List all available Alpha Vantage tools before choosing one to inspect",
  "params": {}
}
```
