## financial-datasets_get_available_crypto_tickers

### What this tool is for
The full list of cryptocurrency tickers available in the financial-datasets server. Use this as a discovery tool before querying crypto price data, to confirm a ticker is supported.

---

### Used parameters

This tool takes no parameters.

---

### JSON input alternatives

```json
{
  "intent": "Discover all supported cryptocurrency tickers before querying prices",
  "params": {}
}
```
