## uspto-patent-mcp_check_api_status

### What this tool is used for
Checks the status of the USPTO data sources used by this server. Use it before starting research or when troubleshooting tool failures.

---

### Used parameters

This tool takes no parameters.

---

### JSON input alternatives

```json
{
  "intent": "Verify the server's data sources before starting patent research",
  "params": {}
}
```

}
