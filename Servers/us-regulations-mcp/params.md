(1) query
 Search query used to find regulation text by topic, concept, or term.
 Required. No default value.

(2) regulations
 Regulation identifiers used to limit results to specific regulations.
 Optional. Default: null.

(3) limit
 Maximum number of results to return.
 Optional. Default: 10.

(4) offset
 Number of matching results to skip for pagination.
 Optional. Default: 0.

(5) regulation
 Regulation identifier used to target a specific regulation when the tool supports regulation-level filtering or retrieval.
 Optional. Default: null.

(6) section
 Section identifier used to retrieve a specific provision within a regulation.
 Required. No default value.

(7) topic
 Topic used to compare how multiple regulations address the same subject.
 Required. No default value.

(8) sector
 Industry sector used to determine which regulations may apply.
 Required. No default value.

(9) subsector
 More specific industry classification used to refine applicability checks.
 Optional. Default: null.

(10) framework
 Control framework used as the source for compliance mappings.
 Required. No default value.

(11) control
 Specific control identifier used to narrow mappings within a framework.
 Optional. Default: null.

(12) sections
 List of section identifiers used when a tool operates on multiple regulation sections at once.
 Required. No default value.

(13) state
 State or jurisdiction name used to narrow results to a specific jurisdiction.
 Optional. Default: null.
