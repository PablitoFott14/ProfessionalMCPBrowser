(1) search_text
 Text used to search for matching FRED series titles and descriptions.
 Optional. Default: null.

(2) search_type
 Search mode used to search by full text or series identifier.
 Optional. Default: null.

(3) tag_names
 Comma-separated list of tag names used to filter matching series.
 Optional. Default: null.

(4) exclude_tag_names
 Comma-separated list of tag names used to exclude matching series.
 Optional. Default: null.

(5) limit
 Maximum number of results or observations to return.
 Optional. Default: 25.

(6) offset
 Number of results or observations to skip.
 Optional. Default: 0.

(7) order_by
 Field used to order the returned series results.
 Optional. Default: null.

(8) sort_order
 Sort direction used for ordered results or observations.
 Optional. Default: null.

(9) filter_variable
 Variable used to filter the search results.
 Optional. Default: null.

(10) filter_value
 Value used for the selected filter variable.
 Optional. Default: null.

(11) series_id
 FRED series identifier used to retrieve data for a specific economic series.
 Required. No default value.

(12) observation_start
 Start date used to limit returned observations.
 Optional. Default: null.

(13) observation_end
 End date used to limit returned observations.
 Optional. Default: null.

(14) units
 Data transformation used for the returned series values.
 Optional. Default: null.

(15) frequency
 Frequency aggregation used for the returned observations.
 Optional. Default: null.

(16) aggregation_method
 Aggregation method used when frequency aggregation is applied.
 Optional. Default: null.

(17) output_type
 Output type used to control how FRED observations are returned.
 Optional. Default: null.

(18) vintage_dates
 Vintage date or dates used for vintage-aware series retrieval.
 Optional. Default: null.

(19) browse_type
 Browse mode used to explore categories, releases, sources, or series collections.
 Required. No default value.

(20) category_id
 Category identifier used for category-based browsing.
 Optional. Default: null.

(21) release_id
 Release identifier used for release-based browsing.
 Optional. Default: null.
