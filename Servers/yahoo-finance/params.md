(1) symbol
 Stock ticker symbol to retrieve data for (e.g., AAPL, GOOGL, TSLA).
 Required. No default value.

(2) count
 Number of news articles to fetch.
 Optional. Default: 10.

(3) query
 Search query used to find stocks, ETFs, or other financial instruments.
 Required. No default value.

(4) entity_type
 Type of entities to retrieve.
 Required. No default value.

(5) sector
 Sector name used to filter the returned entities.
 Optional. Default: "".

(6) period
 Period selector used by the tool to determine the requested time scope or reporting cadence.
 Optional. Default: 1y.

(7) interval
 Data interval used for the historical price series.
 Optional. Default: 1d.

(8) option_type
 Type of options to retrieve.
 Optional. Default: both.

(9) date
 Specific date used by the tool when date-based retrieval is supported.
 Optional. Default: null.
