## alpaca_get_account_info

### What this tool is for
Retrieves current account information from Alpaca, including account ID, status, buying power, cash, equity, and PDT (Pattern Day Trader) status. Use this when the user needs a snapshot of their account's financial state or wants to verify account standing before placing trades.

---

### Used parameters

This tool takes no parameters.

---

### JSON input alternatives

```json
{
  "tool": "alpaca_get_account_info",
  "intent": "Check current account balance, buying power, and trading status",
  "params": {}
}
```
