## alpaca_get_all_positions

### What this tool is for
Retrieves all current open positions in the Alpaca portfolio. Returns each position's symbol, quantity, market value, average entry price, current price, and profit/loss. Use this when the user wants a full view of their active holdings.

---

### Used parameters

This tool takes no parameters.

---

### JSON input alternatives

```json
{
  "tool": "alpaca_get_all_positions",
  "intent": "Get a full overview of all current portfolio positions including P/L",
  "params": {}
}
```
