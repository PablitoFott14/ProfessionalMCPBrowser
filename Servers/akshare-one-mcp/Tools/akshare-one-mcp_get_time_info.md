## akshare-one-mcp_get_time_info

### What this tool is for
The current time context, including ISO-formatted time, a timestamp, and the last trading day. It is useful when the user needs a quick market-time reference before making sense of real-time or recent market data.

---

### Used parameters

This tool takes no parameters.
