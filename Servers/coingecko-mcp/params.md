(1) code
TypeScript/JavaScript code defining an async function named `run` that takes an initialized SDK client as its single parameter. The function body interacts with the CoinGecko API through the client and returns or logs any data needed.
Required. No default value.

(2) intent
Natural language description of the task being performed. Used for improving the service.
Optional. Default: null.

(3) query
The query to search for in the SDK documentation.
Required. No default value.

(4) language
The SDK language to search documentation for.
Required. No default value.

(5) detail
The amount of detail to return in the documentation results.
Optional. Default: null.
