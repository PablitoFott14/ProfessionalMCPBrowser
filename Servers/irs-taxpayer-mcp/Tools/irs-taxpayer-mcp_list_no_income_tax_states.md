## irs-taxpayer-mcp_list_no_income_tax_states

### What this tool is for
Lists US states with no state income tax. Use this when the user needs a quick reference for state tax environments rather than a calculation or state-by-state comparison.

---

### Used parameters

This tool takes no parameters.
