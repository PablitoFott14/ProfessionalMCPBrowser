## open-legal-compliance-mcp_search_uk_legislation

### What this tool is used for
Searches UK legislation by query string. Use it when researching British statutory law or locating Acts of Parliament and statutory instruments relevant to a compliance topic.

---

### Used parameters

**(1) query: Required**
Default: No default
Search query for UK legislation content.

---

### JSON input alternatives

```json
{
  "intent": "Search UK legislation for provisions on a compliance topic",
  "params": {
    "query": "data protection personal data"
  }
}
```

```json
{
  "intent": "Search UK legislation for employment law provisions",
  "params": {
    "query": "employee rights unfair dismissal"
  }
}
```
