(1) query
Search query string.
Required. No default value.

(2) title
Specific US Code title to search within (e.g., "Title 15").
Optional. No default value.

(3) court
Court ID filter (e.g., "scotus", "ca9").
Optional. No default value.

(4) language
Language code for the response.
Optional. Default: en.

(5) state
State code to search within.
Required. No default value.
Allowed: ca, ny, il.

(6) congress
Congress number to filter by (e.g., 118).
Optional. No default value.

(7) perPage
Number of results to return per page.
Optional. No default value.

(8) cik
SEC Central Index Key (10 digits).
Required. No default value.

(9) jurisdiction
State abbreviation to filter by (e.g., "ca").
Optional. No default value.

(10) databaseId
CanLII database identifier to search within.
Optional. Default: csc-scc.

(11) type
Category of FDA search to perform.
Required. No default value.
Allowed: drug, device, food.
