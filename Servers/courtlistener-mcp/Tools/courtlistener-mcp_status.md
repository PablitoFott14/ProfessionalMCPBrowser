## courtlistener-mcp_status

### What this tool is used for
Checks the status of the CourtListener MCP server. Use it to confirm the server is reachable and retrieve system metrics and service information before making other requests.

---

### Used parameters

No parameters.

{
  "intent": "Check server status and retrieve system metrics",
  "params": {}
}
